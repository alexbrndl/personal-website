## Le Projet

Ce portfolio a été conçu avec une approche minimaliste, mettant l'accent sur le contenu et la typographie. L'objectif était de créer un espace personnel qui reflète ma vision du design : simple, fonctionnel et élégant.

![fitness-cover](/images/fitness-cover.jpg "test")

## Stack Technique

Le site est construit avec **Nuxt 4** pour profiter du rendu hybride et de l'écosystème Vue.js. Le styling repose sur du CSS classique avec des variables custom et un design system cohérent.

Les contenus sont gérés via **Nuxt Content v3**, permettant d'écrire les projets et articles en Markdown avec des composants Vue intégrés.

## Design System

Le design system repose sur deux typographies complémentaires :

- **Galnoy** pour les titres, apportant du caractère
- **Geist** pour le corps de texte, optimisé pour la lisibilité

Le thème adaptatif supporte trois modes : système, clair et sombre.
